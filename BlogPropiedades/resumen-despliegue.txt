RESUMEN DE DESPLIEGUE: BLOG COMO SUBDOMINIO DE REALESTATEGOZAMADRID.COM

Hemos preparado tu aplicación para ser desplegada como subdominio de realestategozamadrid.com utilizando AWS Amplify y Cloudflare. A continuación, se detallan los pasos que debes seguir:

1. DESPLIEGUE EN AWS AMPLIFY:
   - Inicia sesión en la consola de AWS: https://aws.amazon.com/console/
   - Busca y selecciona "AWS Amplify"
   - Crea una nueva aplicación seleccionando "Implementar sin Git"
   - Configura la aplicación con el nombre "blog-gozamadrid"
   - Sube el archivo ZIP que hemos creado (blog-gozamadrid.zip)
   - Configura el dominio personalizado: blog.realestategozamadrid.com
   - Configura las variables de entorno según el archivo aws-amplify-config.txt
   - Configura las redirecciones para SPA según el archivo aws-amplify-config.txt
   - Anota la URL de Amplify generada (será algo como d1234abcdef.amplifyapp.com)

2. CONFIGURACIÓN DE CLOUDFLARE:
   - Inicia sesión en tu cuenta de Cloudflare: https://dash.cloudflare.com/
   - Selecciona el dominio realestategozamadrid.com
   - Agrega un registro CNAME:
     * Nombre: blog
     * Destino: [URL de Amplify anotada en el paso anterior]
     * Proxy: Activado
   - Configura SSL/TLS en modo "Full"
   - Crea una regla de página para redirigir HTTP a HTTPS
   - Configura la caché según el archivo cloudflare-dns-config.txt

3. VERIFICACIÓN:
   - Espera a que se propague el DNS (puede tardar hasta 24 horas)
   - Verifica que el subdominio blog.realestategozamadrid.com funcione correctamente
   - Comprueba que las redirecciones y la navegación SPA funcionen correctamente
   - Verifica que la conexión con el backend funcione correctamente

4. MANTENIMIENTO:
   - Para futuras actualizaciones, simplemente construye la aplicación con "npm run build"
   - Comprime la carpeta dist en un archivo ZIP
   - Sube el nuevo ZIP a AWS Amplify

ARCHIVOS DE CONFIGURACIÓN CREADOS:
- amplify.yml: Configuración para AWS Amplify
- amplify.json: Configuración del proyecto para Amplify
- redirects.json: Configuración de redirecciones para SPA
- cloudflare-dns-config.txt: Instrucciones detalladas para configurar Cloudflare
- aws-amplify-config.txt: Instrucciones detalladas para configurar AWS Amplify

ARCHIVOS MODIFICADOS:
- .env: Actualizado para apuntar al backend de producción
- vite.config.js: Optimizado para producción
- package.json: Actualizado con dependencias compatibles
